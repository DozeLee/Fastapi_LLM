（1）主要用途
  解锁设备唤醒屏幕后手指按压 Touch ID 即可解锁（无
 需输密码）
  验证操作下载 AppApple Pay 支付修改系统设置
 时按压 Touch ID 快速验证
  替代密码部分第三方软件（如 Chrome微信）可设置
 用 Touch ID 登录
（2）添加 / 删除指纹
  打开系统设置 点击左侧触控 ID 与密码
  输入管理员密码进入设置界面
  添加指纹点击添加指纹按提示按压手指（确保指
 纹覆盖传感器）完成后命名（如右手食指）
  删除指纹点击已录入的指纹选择移除指纹即可
单指轻击选择文件 / 按钮
  双指轻击右键菜单（等同于鼠标右键）
  双指上下滑动滚动页面
  双指捏合缩放图片 / 网页
  窗口控制
  左上角红色按钮关闭当前窗口
  黄色按钮最小化窗口到 Dock 栏
  绿色按钮全屏 / 还原窗口（长按可选择 tile 窗
 口分屏）

3.1 Touch ID 与密码
清洁用干燥软布擦拭 Touch ID 区域（避免油污水渍影
 响识别）
  重录删除原有指纹后重新录入（确保手指干燥录入时
 覆盖传感器边缘）
  重启重启设备后重试（部分系统 bug 可通过重启修
 复）

 的程序
